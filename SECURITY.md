# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it to us by emailing `security@example.com`. We will respond to your report within 48 hours.

Please include the following information in your report:

*   A description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   The version of the project you are using.
*   Any other relevant information.

We appreciate your help in keeping our project secure.
